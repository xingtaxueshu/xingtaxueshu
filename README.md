# 星塔学术 · XINGTA Academic

We build practical AI workflows for academic research.

Our focus is narrow on purpose: literature reviews, research design, methodology reasoning, and writing systems that keep the researcher's judgment in charge.

## What We Work On

- **Literature review workflows**: question refinement, review modules, contradiction mapping, and research-gap discovery.
- **Academic writing systems**: reusable skills and prompts that help researchers organize evidence instead of generating unsupported prose.
- **Research-method tooling**: small agent workflows for method comparison, proposal logic, and structured reading.
- **AI for social science research**: practical experiments with Claude Code, Codex, and agent skills.

## Featured Project

### [Literature Review Skills](https://github.com/xingtaxueshu/literature-review-skills)

Five open agent skills for writing stronger literature reviews:

1. refine a broad topic into a real review question;
2. decompose the question into review modules;
3. compare research methods by what they can and cannot answer;
4. map conflicting findings without forcing fake agreement;
5. locate a concrete and defensible research gap.

The project is designed for Claude Code and Codex users, but the workflow can also be used without any AI tool.

Repository: https://github.com/xingtaxueshu/literature-review-skills
